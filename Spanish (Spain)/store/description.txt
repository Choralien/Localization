El mando para ordenador que más funciones incluye. Convierta su dispositivo en un mando a distancia universal por Wi-Fi o Bluetooth para PC con Windows. También compatible con Mac y Linux. Controle sus programas favoritos, el ratón y el teclado.

Servidor: https://www.unifiedremote.com

“Una herramienta fantástica para quienes quieran tener un mando a distancia para su ordenador, para ver la tele o escuchar música”. – Gizmodo

Funciones principales:
• Configuración sencilla de servidor y app que funciona de verdad. 
• Detección automática del servidor para encontrar fácilmente los servidores de su red. 
• Protección del servidor con contraseña y cifrado para mayor seguridad. 
• Compatible con control del ratón de un toque y toque múltiple. 
• Active fácilmente su servidor con la opción de encender con LAN. 
• Incluye temas de color claro y oscuro. 
• El servidor está disponible para Windows, Mac y Linux. 
• Control de otros dispositivos como Raspberry Pi y Arduino Yún. 
• 18 mandos gratuitos
